# jcv

Web app to render a CV from a JSONResume file.

Built with Next.js and shadcn/ui, data stored on Upstash, deployed on Vercel.

Heavily inspired by [Bartosz Jarocki's cv](https://github.com/BartoszJarocki/cv).
